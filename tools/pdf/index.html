📄 পিডিএফ টুলস (Advanced PDF Tools)

সম্পূর্ণ ফ্রি এবং দ্রুত অনলাইন পিডিএফ সリューション প্ল্যাটফর্ম।

🔗

১. পিডিএফ জোড়া দিন

একাধিক PDF ফাইলকে একত্রে একটিতে যুক্ত করুন

✂️

২. পিডিএফ আলাদা করুন

একটি PDF থেকে নির্দিষ্ট পেজ আলাদা করুন

📝

৩. PDF to Word

PDF ফাইল থেকে টেক্সট এক্সট্রাক্ট করে ডকুমেন্ট তৈরি করুন

📄

৪. Word to PDF

যেকোনো টেক্সট বা নোটিশ লিখে সরাসরি PDF তৈরি করুন

🗜️

৫. পিডিএফ সাইজ কমান

PDF ফাইলের কোয়ালিটি ঠিক রেখে সাইজ ছোট করুন

🖼️

৬. Images to PDF

JPG, PNG বা অন্যান্য ছবি থেকে PDF তৈরি করুন

📸

৭. PDF to Images

PDF-এর পেজকে হাই-কোয়ালিটি ছবিতে রূপান্তর করুন

🔄

৮. Rotate PDF

PDF-এর পেজ ৯০°, ১৮০° বা ২৭০° ঘুরিয়ে সাজান

🔓

৯. Unlock PDF

পাসওয়ার্ড জানা থাকলে PDF-এর নিরাপত্তা লক সরিয়ে ফেলুন

🔒

১০. Protect PDF

নিজের পছন্দের পাসওয়ার্ড দিয়ে PDF সুরক্ষিত করুন

🔗 পিডিএফ জোড়া দিন (PDF Merge)

একত্রে যুক্ত করার জন্য দুই বা তার বেশি পিডিএফ ফাইল নির্বাচন করুন।

✂️ পিডিএফ আলাদা করুন (PDF Split)

নির্দিষ্ট পেজ নম্বর আলাদা করে নতুন পিডিএফ ফাইল তৈরি করুন।

📝 PDF to Word Converter

পিডিএফ ফাইল আপলোড করে ভেতরের টেক্সটগুলোকে এডিটেবল ডক টেক্সট হিসেবে নামিয়ে নিন।

⚠️ GitHub Pages-এর সীমাবদ্ধতার কারণে শুধুমাত্র Text Extract করা সম্ভব। নিখুঁত Word Conversion-এর জন্য Server Version প্রয়োজন।

📄 Word / Text to PDF Creator

যেকোনো টেক্সট বা ডকুমেন্ট টাইপ করে সরাসরি হাই-কোয়ালিটি পিডিএফে রূপান্তর করুন।

🗜️ পিডিএফ সাইজ কমান (PDF Compress)

ফাইলের অভ্যন্তরীণ মেটাডেটা অপ্টিমাইজ করে সাইজ ছোট ও হালকা করুন।

🖼️ Images to PDF Converter

এক বা একাধিক ছবি (JPG, PNG, WEBP) আপলোড করে একটি কম্বাইন্ড পিডিএফ তৈরি করুন।

📸 PDF to Images Converter

পিডিএফ-এর সব পেজ আলাদা আলাদা হাই-কোয়ালিটি ছবিতে রূপান্তর করে ডাউনলোড করুন।

🔄 Rotate PDF Pages

পিডিএফ পেজগুলোকে নির্দিষ্ট ডিগ্রিতে ঘুরিয়ে সোজা করুন।

🔓 Unlock Secured PDF

পাসওয়ার্ড জানা থাকা লক ফাইলের সিকিউরিটি সরিয়ে ওপেন করুন।

🔒 Protect PDF with Password

যেকোনো সাধারণ পিডিএফ ফাইলে স্ট্রং পাসওয়ার্ড সেট করে সিকিউর করুন।
